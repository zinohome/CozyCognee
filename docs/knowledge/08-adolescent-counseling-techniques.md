# 青少年咨询技巧和方法

## 一、建立关系技巧

### 1.1 建立信任

#### 信任的重要性

**咨询的基础**
- 信任是咨询的基础
- 没有信任，咨询难以进行
- 信任需要时间建立
- 信任需要持续维护

**青少年的特点**
- 青少年可能不信任成人
- 可能对咨询有抵触
- 可能害怕被评判
- 需要更多时间建立信任

#### 建立信任的方法

**真诚一致**
- 做真实的自己
- 不伪装，不做作
- 言行一致
- 表里如一

**尊重接纳**
- 尊重青少年
- 无条件接纳
- 不评判
- 不批评

**理解共情**
- 深入理解青少年
- 感受青少年的感受
- 表达理解
- 验证理解

**保密承诺**
- 明确保密承诺
- 解释保密例外
- 遵守保密原则
- 保护隐私

**实践示例**：
```
咨询师：我知道第一次来咨询可能会感到紧张或不信任。这是很正常的。我想告诉你，我会对你在这里说的一切保密，除非涉及安全问题。我不会评判你，我会尽力理解你。我们可以慢慢来，不用着急。
来访者：真的吗？你不会告诉我父母？
咨询师：除非涉及安全问题，我不会告诉你的父母。这是你的隐私，我会保护。
```

### 1.2 处理阻抗

#### 阻抗的表现

**直接阻抗**
- 拒绝来咨询
- 拒绝说话
- 拒绝参与
- 直接对抗

**间接阻抗**
- 迟到
- 早退
- 沉默
- 转移话题

**被动阻抗**
- 表面配合
- 实际不参与
- 不完成作业
- 不改变

#### 阻抗的原因

**不信任**
- 不信任咨询师
- 不信任咨询
- 害怕被伤害
- 害怕被评判

**害怕**
- 害怕改变
- 害怕面对问题
- 害怕失去控制
- 害怕被暴露

**被动**
- 被动来咨询
- 不是自愿的
- 缺乏动机
- 缺乏准备

#### 处理阻抗的方法

**理解阻抗**
- 理解阻抗的意义
- 阻抗是正常的
- 阻抗是保护机制
- 不评判阻抗

**建立信任**
- 通过建立信任减少阻抗
- 表达理解和关心
- 不强迫
- 给予时间

**灵活应对**
- 灵活应对阻抗
- 不强迫参与
- 寻找切入点
- 适应青少年的节奏

**处理原因**
- 处理阻抗的原因
- 处理不信任
- 处理害怕
- 处理被动

**实践示例**：
```
咨询师：我注意到你今天似乎不太想说话。能告诉我发生了什么吗？
来访者：（沉默）
咨询师：没关系，不想说也可以。我们可以做点别的，比如画画或者玩游戏。
来访者：真的可以吗？
咨询师：当然。我们可以用任何你觉得舒服的方式。你想做什么？
来访者：我想画画...
咨询师：好，那我们画画。你可以画任何你想画的。
```

## 二、沟通技巧

### 2.1 与青少年沟通

#### 倾听技巧

**积极倾听**
- 全神贯注地倾听
- 不仅听内容，还听情感
- 不仅听说的，还听未说的
- 理解青少年的世界

**非语言关注**
- 保持眼神接触（适度）
- 身体前倾
- 点头回应
- 避免分心

**理解回应**
- 通过回应表达理解
- 验证理解是否准确
- 帮助青少年更深入理解自己
- 促进探索

**实践示例**：
```
来访者：我觉得没有人理解我...
咨询师：你感到孤独，觉得没有人真正理解你的感受和处境。
来访者：是的，我从来没有被真正理解过...
咨询师：这种不被理解的感觉，让你感到深深的孤独和痛苦。能告诉我更多吗？
```

#### 提问技巧

**开放式提问**
- 鼓励详细表达
- 探索问题
- 收集信息
- 促进思考

**封闭式提问**
- 确认信息
- 聚焦问题
- 需要具体答案时使用

**假设性提问**
- 探索可能性
- 促进思考
- 帮助看到不同选择

**避免引导性提问**
- 避免暗示答案
- 保持中立
- 让青少年自己探索

**实践示例**：
```
咨询师：能告诉我你和朋友之间发生了什么吗？（开放式）
来访者：我们吵架了...
咨询师：你们为什么吵架？（开放式）
来访者：因为意见不合...
咨询师：如果你们能够坐下来好好谈谈，会发生什么？（假设性）
来访者：可能会好一些...
咨询师：你愿意尝试吗？（封闭式）
```

#### 表达技巧

**共情表达**
- 准确表达对青少年感受的理解
- 帮助青少年认识自己的感受
- 验证理解是否准确
- 促进深入探索

**反馈技巧**
- 提供观察和反馈
- 帮助青少年看到自己
- 促进自我觉察
- 支持成长

**实践示例**：
```
咨询师：我注意到当你说到这件事时，你的声音变小了，身体也缩起来了。能告诉我你现在的感受吗？
来访者：我感到害怕...
咨询师：害怕什么？
来访者：害怕被评判...
咨询师：在这里，你不会被评判。我会尽力理解你。
```

- 不泄露不必要的信息
- 征得同意后分享
- 平衡保密与安全

**尊重自主**
- 尊重青少年自主

**实践示例**：
```

咨询师：我理解您的担心。同时，我也想了解您孩子的感受。在咨询中，我会保护他的隐私，但涉及安全问题时，我会告知您。我们可以一起支持他。

**识别非理性思维**

**挑战非理性思维**

**重构理性思维**
- 用更理性的思维替代
- 形成平衡的观点
- 建立新的思维模式

**实践示例**：
```
来访者：如果我这次考试不及格，我就是一个失败者。

来访者：有...
咨询师：那时候你是一个失败者吗？
来访者：不是...
咨询师：所以，一次考试不及格，不代表你就是一个失败者，对吗？
来访者：对...

## 五、实践要点

### 5.1 技术选择的依据

- **根据问题类型选择**：不同问题选择不同技术
- **根据青少年特点选择**：适应青少年的特点
- **根据发展阶段选择**：适应发展阶段
- **根据咨询阶段选择**：不同阶段选择不同技术

### 5.2 技术的整合

- **以关系为基础**：所有技术都建立在良好的关系基础上
- **灵活运用**：不局限于单一技术，灵活整合
- **以效果为导向**：选择最有效的技术
- **持续调整**：根据效果持续调整

### 5.3 注意事项

- **不要机械使用**：技术要灵活运用
- **不要忽视关系**：关系比技术更重要
- **不要忽视发展**：要从发展角度使用技术
- **不要忽视安全**：安全是第一位的
